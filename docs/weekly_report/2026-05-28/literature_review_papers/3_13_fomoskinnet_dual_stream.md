## 우리 연구에서의 위치

ISIC 2024 SLICE-3D subset을 사용해 non-dermoscopic lesion image classification을 수행한 최신 dual-stream image model이다. focal modulation과 local feature branch를 결합하므로 image-only baseline 아이디어로 참고할 수 있지만, balanced subset을 사용했다는 점을 원본 challenge setting과 분리해야 한다.

---

## 목표와 기여

low-quality/non-dermoscopic skin lesion image에서 local texture와 global context를 동시에 학습하는 FoMoSkinNet을 제안한다.

## Dataset 정보

- Dataset: ISIC 2024 SLICE-3D subset
- 구성: benign 15,000장, malignant 15,000장
- 총 image 수: 30,000개
- Split: 80% train, 20% test, train의 10% validation

## Imbalance 처리

benign/malignant 15,000:15,000 balanced dataset을 구성해 binary class imbalance를 직접 완화한다. 이는 원본 ISIC 2024의 400,666:393 imbalance를 유지한 setting이 아니다.

## Tabular model

별도 tabular model은 없다.

## Image model

global feature는 FocalNet branch가 담당하고, local feature는 grouped convolution, channel shuffle, multi-scale dilated depthwise convolution을 포함한 LFNet branch가 담당한다.

## Fusion 방식

FocalNet branch와 LFNet branch의 complementary feature를 hybrid feature fusion으로 결합한다. image-image feature fusion이며 metadata fusion은 아니다.

## 평가 지표

accuracy, F1-score, specificity, precision, sensitivity/recall, ROC, AUC, confusion matrix를 사용한다.

## 평가 결과

ISIC 2024 balanced subset에서 accuracy 98.85%, precision 99.46%, recall/sensitivity 98.23%, F1-score 98.84%, specificity 99.47%를 보고한다.

## ISIC2024 연구 시사점

- non-dermoscopic 3D-TBP tile image에서 global/local feature를 분리하는 image branch 설계 참고가 가능하다.
- ISIC 2024를 직접 사용했다는 점은 유용하지만, balanced subset 구성 때문에 Kaggle original distribution과 직접 비교하면 안 된다.
- image-tabular fusion 연구에서는 image-only strong baseline 후보 중 하나로 언급할 수 있다.

## 추가 논의/주의점

- malignant 15,000장을 어떻게 구성했는지 원본 label 수와 비교해 sampling/augmentation 여부를 확인해야 한다.
- pAUC > 80% TPR가 아니라 일반 classification metric 중심이다.
- metadata를 쓰지 않으므로 multimodal fusion의 직접 근거는 아니다.
