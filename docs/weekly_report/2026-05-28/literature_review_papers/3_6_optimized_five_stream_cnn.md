## 우리 연구에서의 위치

multi-stream image feature와 GAN 기반 class balancing을 결합한 melanoma detection 논문이다. 다만 여러 dataset에서 매우 높은 accuracy를 보고하므로, ISIC 2024 연구에서는 직접적인 SOTA baseline이라기보다 전처리, feature stream, synthetic balancing의 보조 참고로 사용하는 것이 안전하다.

---

## 목표와 기여

hair/noise/artifact와 class imbalance 문제를 전처리하고, lesion color, edge, texture, frequency, gradient 정보를 multi-stream CNN으로 결합해 melanoma detection 정확도를 높이는 방법을 제안한다.

## Dataset 정보

- HAM10000
- ISIC 2024
- ISIC 2017
- ISIC 2016

여러 dermoscopy 및 skin lesion image dataset에서 성능을 보고한다.

## Imbalance 처리

GAN 기반 이미지 생성을 통해 class balance를 맞춘다. hair removal, CNN denoising, enhancement, contrast adjustment 등 전처리도 함께 적용한다.

## Tabular model

별도 tabular model은 없다.

## Image model

ULBP-CVA, multi-block ULBP-NP, Gabor/gradient feature map 등을 입력으로 사용하는 optimized five-stream CNN(OFSCNN)을 제안한다.

## Fusion 방식

lesion color, edge, texture, local-spatial frequency, multi-oriented gradient feature stream을 CNN 내부에서 결합한다. 이는 image feature stream fusion이며 image-tabular fusion은 아니다.

## 평가 지표

accuracy, precision, recall, F1-score, 5-fold cross-validation을 사용한다.

## 평가 결과

HAM10000 99.8%, ISIC 2024 99.9%, ISIC 2017 99.62%, ISIC 2016 99.6% detection accuracy를 보고하며, 기존 SOTA 대비 recall, precision, F1 향상을 주장한다.

## ISIC2024 연구 시사점

- lesion-specific handcrafted-like feature stream과 CNN feature를 결합하는 image-only baseline 아이디어로 참고할 수 있다.
- GAN/diffusion synthetic augmentation을 minority class balancing 후보로 논의할 때 참고 가능하다.
- metadata fusion 근거는 아니므로 multimodal model justification에는 제한적으로 사용해야 한다.

## 추가 논의/주의점

- ISIC 2024의 원본 malignant prevalence를 유지했는지, balanced subset을 썼는지 확인이 필요하다.
- 매우 높은 accuracy는 pAUC > 80% TPR setting과 직접 비교하지 않는다.
- GAN 생성 이미지가 train/test leakage나 artifact overfitting을 만들 수 있으므로 train-only 조건에서는 positive train fold 내부에서만 생성해야 한다.
