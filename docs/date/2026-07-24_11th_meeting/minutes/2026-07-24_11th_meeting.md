## 1. 해결하려는 문제와 연구 가설

**문제.** 피부과 FM(PanDerm)을 저자원 out-of-domain(안저·구강)으로 옮길 때 **적응 전략**(어느 레이어를 쓸지 · 얼마의 파라미터 용량을 줄지)을 골라야 한다. 표준 관행은 val/test 성능으로 고르는 것이다. **그런데 이 도메인의 평가셋은 그 선택을 감당할 크기가 아니다.** 우리 데이터에서 OLP(test n=10)는 CV 0.69 ↔ test 0.30이고, kNN head는 test 0.70이 나왔으나 CI[0.4, 1.0]로 baseline과 구분 불가였다 — **같은 노이즈가 한쪽엔 0.30, 다른 쪽엔 0.70을 준다.** 새로 확보한 SMART-OM도 동일(암 20장·OPMD 125장). 즉 **데이터를 더 구해도 사라지지 않는 이 도메인의 구조적 문제**이며, 선택 기준 자체가 노이즈다.

**연구 가설 (H).**

> **저자원·불균형 의료 전이에서 test 성능 기반 전략 선택은 붕괴하지만, 학습 없이 계산하는 전이성 지표(TE)는 동일한 소표본에서도 최적 적응 전략을 안정적으로 예측한다.**

| | 세부 가설 | 현재 상태 |
| --- | --- | --- |
| **H1** (진단) | test 기반 선택은 n이 작아지면 순위가 무작위에 가까워진다 | 🟡 부분 확인 (OLP 사례 1건) |
| **H2** (처방) | TE 지표(LogME 등)는 같은 n에서 test보다 안정적으로 최적 전략을 예측한다 | 🔴 미검증 — **W11 핵심** |
| **H3** (기전) | 도메인 거리 ↑ → 최적 레이어 얕아지고 필요 용량 ↑ | 🟡 부분 확인, ⚠️D3 충돌 있음 |

**⚠️ D1 — 가설 확정 필요.** 위 H로 갈지, 대안(H3를 메인으로: "도메인 거리 기반 적응 전략 선택")으로 갈지. 본인 의견은 §3-D 참조.

## 2. 방법론과 실험 계획 (우선순위 순)

**방법론 근거.** 문제(소표본 선택 붕괴)는 의료 ML에서 이미 보고된 현상이고, 처방(TE)은 성숙한 계열이 있다. **둘을 잇는 것이 본 연구의 위치**다. TE는 fine-tuning 없이 forward pass 한 번으로 전이성을 추정하므로, 애초에 "큰 test 없이 고르기"가 목적 → 10차 피드백의 "추가 데이터 확보 없이 방법론적 완화"에 정확히 부합.

| 순위 | 실험 | 가설 내 위치 | 근거 레퍼런스 | 비용 |
| --- | --- | --- | --- | --- |
| **P0** | **소표본 선택 붕괴 정량화** — test를 n=10~554로 subsample하며 6방법 순위 안정성(Kendall τw) 측정 | H1 증명 (문제의 존재) | Kim & Cho 2021 (radiomics 단일 split 불안정) | 학습 0, 기존 예측 재사용 |
| **P1** | **TE 적용** — LogME/LEEP를 24레이어 × 6방법에 계산, 실측 성능과 순위 상관 | H2 증명 (처방의 유효성) | You+ ICML'21 (LogME), Nguyen+ ICML'20 (LEEP), 음성 레이어 선택 τ=0.87 | forward만, GPU 가벼움 |
| **P2** | **TE의 표본 민감도** — n을 줄이며 TE 순위 예측이 붕괴하는 지점 탐색, P0 곡선과 겹쳐 그림 | **H2의 승부처** (TE가 test보다 나은가) | LogME의 marginal likelihood = overfitting 면역 주장 검증 | 위와 동일 |
| **P3** | **불균형 대응 프로토콜** — class-weighted/focal/logit-adjust × 다중 시드 + bootstrap CI | 10차 피드백 5-2 반영, 성능 보고 표준화 | 교수님 피드백 5-2, 의료영상 평가지표 pitfalls (2025) | 재학습 필요 |
| **P4** | **SMART-OM 3단계 외부검증** (P3 프로토콜 적용) | H 일반화 | 10차 §2-4 매핑 리포트 | 설계 완료·실행 대기 |

**의도적 배제.** LoRA 보조헤드/마진(10차 결정 유지), OLP 데이터 추가 확보(10차 피드백 5-1).

**⚠️ D2 — 도메인 추가 여부.** H3는 "도메인 거리 → 전략"의 **예측 관계** 주장인데 현재 도메인이 **2개뿐**이라 관계를 주장할 수 없음(점 2개). GI 내시경/OCT 추가(원 로드맵) vs 주장을 case study로 하향 중 선택 필요.

## 3. 실험 결과에 대한 본인 의견

### A. OLP 진단·개입 (10차 §2-1b, §2-2)

- **긍정** — "test 기반 선택이 붕괴한다"의 **직접 증거**를 확보했다. 특징은 선형 분리 가능(모든 쌍 ≥0.81)인데 test만 0.30 → 병목이 표현이 아니라 평가임을 분리해냈다. **이게 H1이자 논문의 출발점이 될 수 있다.**
- **부정** — 개입(head 교체)으로 test 성능을 못 올렸다. "성과"로는 약하다.
- **결정** — 이 실패를 결과가 아니라 **문제 정의(§1)**로 재배치. (10차 피드백 5-1과 일치)

### B. PEFT 6방법 스펙트럼 (10차 §2-3)

- **긍정** — 도메인 거리와 필요 용량의 스펙트럼을 확보(aptos는 LoRA 이상 필요, Oral은 LN-tuning 0.04%로 baseline 동률). **P1의 ground-truth로 그대로 쓸 수 있다.**
- **부정** — "LoRA ≈ Full FT, 파라미터 0.4%"는 원 LoRA 논문의 핵심 결과 재현이라 **그 자체로는 novelty가 없다.** 데이터셋 2개 벤치마킹으로 논문을 세우긴 어렵다.
- **결정** — 최종 결과가 아니라 **TE 검증용 정답표**로 역할 변경.

### C. 레이어 스윕 (10차 §③ Fig c)

- **긍정** — 레이어별 전이 패턴을 확보, aptos 중간층(L7–14) 우위는 명확.
- **부정** — 재도출 시 **두 데이터셋 모두 val 최적층 = L11** → "Oral=마지막층" 서사와 충돌. H3의 근거가 흔들린다.
- **결정** — ⚠️D3.

### D. ⚠️ 교수님 피드백이 필요한 결정 4건

| | 결정 | 본인 의견 |
| --- | --- | --- |
| **D1** | 연구 가설 확정 (H vs H3-메인) | **H 권장.** H3는 도메인 2개로 주장 불가(D2)이고 L11 충돌(D3)이 있음. 반면 H는 이미 확보한 실패(OLP)를 문제로, 기존 실험을 정답표로 재활용하며 새 데이터가 필요 없음 |
| **D2** | 도메인 추가(GI/OCT) 여부 | H로 가면 **불필요** (도메인 2개로도 TE 검증 가능). H3로 가면 3–4개 필수 |
| **D3** | H3(도메인 거리→레이어) 유지/폐기 | H 하위 근거로 **격하** 권장. 단독 주장은 위험 |
| **D4** | 목표 venue·투고 시점 | 미정 — scope(full vs workshop)에 따라 D2가 갈림 |

**⚠️ 선행연구 리스크 (정직하게).** "TE로 레이어/PEFT를 고른다"는 아이디어 자체는 이미 존재한다 — PEFTDiff(ICCV 2025)가 PEFT 선택에, 음성 분야가 레이어 선택에 TE를 썼다. **남은 빈틈은 "소표본·불균형 의료 환경에서 TE가 test보다 실제로 안정적인가"** — TE 논문들은 대개 큰 벤치마크에서 검증됐지, TE가 가장 필요한 저자원 환경에서의 신뢰성은 덜 다뤄졌다. P2가 이 빈틈을 정조준한다. 이 빈틈이 충분한지 판단에 피드백 요청.
