### 📊 [분석] aptos2019 클래스 가중치 적용 & 샘플 영향도 [panderm_class_weight_sample_influence_aptos2019_20260708.ipynb](../../../../notebooks/PanDerm/Linear%20Evaluation/panderm_class_weight_sample_influence_aptos2019_20260708.ipynb)

- [x] Linear Evaluation에 클래스 가중치 손실 함수(Weighted CE) 적용 재실험
> `WeightedLogisticRegression`(원본 `LogisticRegression` 상속, 손실만 `CrossEntropyLoss(weight=...)`로 교체)을 노트북에서 자체 정의해 실험. 가중치는 `compute_class_weight("balanced", ...)` 기준(no_dr 0.41 / mild 1.98 / moderate 0.73 / severe 3.80 / proliferative_dr 2.49)
- [x] 결과 비교: 기존 vs 가중치 적용 (Balanced Accuracy / Macro AUPR / 소수 클래스 Recall)
> Balanced Accuracy 0.633→**0.671**(+0.038), Severe Recall 0.367→**0.500**(+0.133), Proliferative Recall 0.444→**0.533**(+0.089) 개선. 단 Accuracy 0.814→0.789(-0.025), Macro AUPR 0.696→0.687(-0.008)는 소폭 하락(주로 moderate Recall 0.79→0.59 저하가 원인) — 상세 수치는 2-1절 표 참고
- [x] LOO 기반 개별 샘플 영향도 분석 (Oral_Diseases와 동일 방법론 적용)
> 전체 학습 샘플 2,562개 대상 풀 LOO 완료(가중치 미적용 baseline 기준, train 셋만 사용 — Oral 선례와 동일 조건). Oral 노트북이 import하던 `sample_influence.py`가 실제로는 존재하지 않아 방법론을 노트북에서 자체 재구현했고, 이산 Recall 대신 연속 확률 기반 influence를 처음부터 기본 지표로 설계(0 비율 이산 52.4% → 연속 1.0%로 양자화 문제 대폭 완화). **핵심 발견: Severe ↔ Proliferative_dr 두 클래스가 서로의 결정경계를 침범하는 양방향 혼동**(각 클래스 예측을 가장 방해하는 학습 샘플이 대부분 상대 클래스) — 당뇨망막병증 등급이 순서형(ordinal)이라 인접 등급끼리 혼동되는 것으로 해석

### 🔬 [연구] Multi-layer 융합 (레이어 탐색) [panderm_multilayer_fusion_layer_exploration_20260708.ipynb](../../../../notebooks/PanDerm/Linear%20Evaluation/panderm_multilayer_fusion_layer_exploration_20260708.ipynb)

- [x] 24개 블록별 CLS 토큰 추출 → 레이어별 probe 곡선 산출 (hook만 사용, 파라미터 학습 0)
> forward hook 으로 24개 블록의 CLS(`out[:,0]`, pre-`self.norm`)를 **한 번의 forward** 로 동시 추출(학습 파라미터 0, `assert trainable==0`). `self.norm` 이 토큰별 LayerNorm 이라 반환값 post-norm 마지막 CLS = 문서화 baseline 과 동일 경로임을 확인(`||norm(prenorm_L23)-postnorm||=0`). 레이어별 StandardScaler(train 통계, 비학습) 후 기존 `eval_linear_probe` 로 probe 곡선 산출. **두 데이터셋 모두 초반(L0~6) 급상승 후 중·후반 포화**, aptos 는 validation BACC 가 **중간층 L11**에서 최고(마지막 L23 은 오히려 낮음) → 중간층이 원거리 도메인에 더 전이 가능하다는 가설과 일치.
- [x] 최적 레이어/조합 선정 (마지막 CLS 단독 vs 중간층 융합 비교)
> **선정은 validation split**(test 누출 방지), 최종 지표는 test 1회 보고 + bootstrap CI. **aptos(원거리)**: val-선정 최적 = **단일 중간층 L11**, test BACC 0.654(baseline 0.621 대비 **+0.033**), 융합(전체24층)도 BACC 0.659·AUPR 0.691(baseline 0.679)로 상회 — 다만 CI(test=554)가 겹쳐 약 1.2σ 수준의 **일관된 경향**. **Oral(근거리)**: 마지막 CLS 로 이미 충분(융합이 BACC 를 넘지 못함, all24 는 AUPR 만 0.798→0.841 개선) — test=54 로 baseline 과 통계적 구분 불가. 상세 수치는 2-2절 표 참고, 상세 곡선/CI 는 노트북 참고.
