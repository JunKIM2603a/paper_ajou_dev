## Strict-Compatible Reading

- 직접 비교 후보: `FusionNetX`, `Multimodal system for skin cancer detection`, `Dual-stage segmentation...`는 image + metadata 조건과 비교적 가깝다.
- 부분 비교 후보: `Automated triage...`, `Hybrid Ensemble...`, `PMM-FL`은 patient context, WB360 proprietary features, diagnosis-informed relabeling, custom dataset 등으로 인해 같은 protocol로만 비교해야 한다.
- feature-audit 후보: `Explainable multimodal AI...`는 XAI 관점으로는 매우 유용하지만 `mel_thick_mm` 같은 pathology-derived feature가 등장하므로 strict baseline 비교에서는 제외하거나 해당 feature 제거 후만 비교해야 한다.
- 논문 주장에 사용할 때는 반드시 `patient-level split`, `train-only preprocessing`, `threshold source`, `metric definition` 확인이 필요하다.
